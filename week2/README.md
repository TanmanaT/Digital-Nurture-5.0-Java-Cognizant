# Week 2 - Spring Core and Maven

This directory contains the hands-on exercises, source code, and configurations implemented during Week 2.

## 📂 Contents

1. **[Spring Core and Maven](./Spring_Core_Maven)**
   - Core Spring concepts focusing on IoC Container, Dependency Injection (Constructor/Setter), Aspect-Oriented Programming (AOP), Annotations, XML/Annotation-based Configurations, Maven Project Setup, and Spring Boot basics.
2. **[Spring Data JPA with Hibernate](./Spring%20Data%20JPA%20with%20Hibernate)**
   - Hands-on exercises and configurations covering ORM, Spring Data JPA CRUD, Derived Query Methods, Entity Relationships (@ManyToOne, @OneToMany, @ManyToMany), HQL/JPQL queries, Native SQL, Criteria Builder, Projections, Auditing, Pagination, and Spring Boot datasource customization.
