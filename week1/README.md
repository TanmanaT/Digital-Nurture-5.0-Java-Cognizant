# Week 1 - Core Development Concepts, Database Programming & Testing

This directory contains the hands-on exercises, source code, and tests implemented during Week 1.

## 📂 Contents

1. **[Engineering Concepts](./Engineering_Concepts)**
   - Core concepts focusing on Algorithms, Data Structures, and Software Design Patterns (Factory, Builder, Singleton, Adapter, Decorator, Proxy, Command, Strategy, Observer, MVC, Dependency Injection).

2. **[PL/SQL Programming](./PL%20SQL%20programming)**
   - Oracle PL/SQL database scripts covering Control Structures, Stored Procedures, Custom Functions, Triggers, Explicit Cursors, Error Handling, and Packages.

3. **[JUnit, Mockito and SL4J](./JUnit_Mockito_SLF4J)**
   - Java unit testing, integration testing, dependency mocking, and logging exercises structured in 7 modules with self-contained Maven configuration.
