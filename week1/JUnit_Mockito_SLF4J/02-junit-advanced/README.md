# Booklet 2: Advanced JUnit Testing Exercises

This booklet covers advanced features of JUnit 5 (Jupiter), such as:
1. **[Exercise 1: Parameterized Tests](./ex1)**: Run the same test with different values.
2. **[Exercise 2: Test Suites and Categories](./ex2)**: Group related tests into suites.
3. **[Exercise 3: Test Execution Order](./ex3)**: Control execution sequence.
4. **[Exercise 4: Exception Testing](./ex4)**: Assert that methods throw expected exceptions.
5. **[Exercise 5: Timeout and Performance Testing](./ex5)**: Enforce test duration limits.
