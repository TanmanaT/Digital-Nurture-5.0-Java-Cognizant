# Booklet 5: Spring Testing Exercises

This booklet covers testing Java applications using Spring Boot Test features:
1. **[Exercise 1: Basic Unit Test for a Service Method](./ex1)**: Unit testing a `@Service` bean without context loading.
2. **[Exercise 2: Mocking a Repository in a Service Test](./ex2)**: Mocking data layer inside service unit tests using `@Mock` or Mockito.
3. **[Exercise 3: Testing a REST Controller with MockMvc](./ex3)**: Testing controller API endpoints using `MockMvc`.
4. **[Exercise 4: Integration Test with Spring Boot](./ex4)**: Writing full-flow integration tests using `@SpringBootTest`.
5. **[Exercise 5: Test Controller POST Endpoint](./ex5)**: MockMvc validation of resource creation via POST endpoints.
6. **[Exercise 6: Test Service Exception Handling](./ex6)**: Testing service layer exception propagation.
7. **[Exercise 7: Test Custom Repository Query](./ex7)**: Database integration tests utilizing custom JPA repository query methods.
8. **[Exercise 8: Test Controller Exception Handling](./ex8)**: Testing global exception handlers (`@ControllerAdvice`) using MockMvc.
9. **[Exercise 9: Parameterized Test with JUnit](./ex9)**: Advanced parameterized inputs on service logic.
