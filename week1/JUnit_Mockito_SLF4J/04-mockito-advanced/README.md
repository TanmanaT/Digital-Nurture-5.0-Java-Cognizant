# Booklet 4: Advanced Mockito Hands-On Exercises

This booklet covers mocking complex dependencies such as databases, external REST APIs, file I/O, and network resources.

## Exercises

1. **[Exercise 1: Mocking Databases and Repositories](./ex1)**: Stubbing repository layer methods to test business logic in service classes.
2. **[Exercise 2: Mocking External Services (RESTful APIs)](./ex2)**: Testing services that depend on external REST client responses.
3. **[Exercise 3: Mocking File I/O](./ex3)**: Stubbing File Readers and Writers to test file processing services.
4. **[Exercise 4: Mocking Network Interactions](./ex4)**: Simulating network calls and connections.
5. **[Exercise 5: Mocking Multiple Return Values](./ex5)**: Mocking multiple consecutive invocations on stubbed components.
