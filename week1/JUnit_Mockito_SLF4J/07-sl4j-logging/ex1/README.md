# Exercise 1: Logging Error Messages and Warning Levels

## Task
Write a Java application that demonstrates logging error messages and warning levels using SL4J.

## Steps
1. Add SL4J API and Logback classic implementation dependencies to `pom.xml`.
2. Create `LoggingExample.java` containing log statements at `error` and `warn` levels.

## Solution Files
- [`pom.xml`](./pom.xml): Maven project descriptor containing SL4J and Logback dependencies.
- [`src/main/LoggingExample.java`](./src/main/LoggingExample.java): Java class utilizing SL4J Logger.
