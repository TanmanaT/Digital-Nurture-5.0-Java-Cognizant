# Booklet 7: Logging using SL4J

This booklet covers basic logging using SL4J API and Logback framework.

## Exercises

1. **[Exercise 1: Logging Error Messages and Warning Levels](./ex1)**: Setting up dependencies and writing basic log statements.
2. **[Exercise 2: Parameterized Logging](./ex2)**: Formatting logging strings with placeholders (`{}`) to optimize performance.
3. **[Exercise 3: Using Different Appenders](./ex3)**: Configuring `logback.xml` to log both to Console and to a file.
