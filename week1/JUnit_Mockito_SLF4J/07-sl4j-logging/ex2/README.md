# Exercise 2: Parameterized Logging

## Task
Write a Java application that demonstrates parameterized logging using SL4J.

## Steps
1. Add SL4J API and Logback dependencies.
2. Create `ParameterizedLoggingExample.java` using placeholders (`{}`) to log dynamic runtime values instead of string concatenation.

## Solution Files
- [`pom.xml`](./pom.xml): Maven project descriptor containing logging dependencies.
- [`src/main/ParameterizedLoggingExample.java`](./src/main/ParameterizedLoggingExample.java): Java class utilizing SL4J logger placeholders.
