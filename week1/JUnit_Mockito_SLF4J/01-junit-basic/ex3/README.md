# Exercise 3: Assertions in JUnit

## Scenario
You need to use different assertions in JUnit to validate your test results.

## Steps
1. Create a test class `AssertionsTest`.
2. Write a test case `testAssertions()` utilizing `assertEquals`, `assertTrue`, `assertFalse`, `assertNull`, and `assertNotNull`.

## Solution Files
- [`pom.xml`](./pom.xml): Maven project descriptor containing the JUnit 4 dependency.
- [`src/test/AssertionsTest.java`](./src/test/AssertionsTest.java): JUnit test class demonstrating various assertion methods.
