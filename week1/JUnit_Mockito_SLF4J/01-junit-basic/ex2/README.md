# Exercise 2: Writing Basic JUnit Tests

## Scenario
You need to write basic JUnit tests for a simple Java class.

## Steps
1. Create a `Calculator` class with methods `add`, `subtract`, `multiply`, and `divide`.
2. Write unit tests for all methods, including validating the exception when dividing by zero.

## Solution Files
- [`pom.xml`](./pom.xml): Maven project descriptor containing the JUnit 4 dependency.
- [`src/main/Calculator.java`](./src/main/Calculator.java): Simple calculator class containing math methods.
- [`src/test/CalculatorTest.java`](./src/test/CalculatorTest.java): Test class with JUnit tests for add, subtract, multiply, and divide.
