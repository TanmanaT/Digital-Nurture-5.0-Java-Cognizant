# Booklet 1: JUnit Basic Testing Exercises

This module covers the basics of unit testing using JUnit 4.

## Exercises

1. **[Exercise 1: Setting Up JUnit](./ex1)**: Creating a basic maven project structure and a dummy test class.
2. **[Exercise 2: Writing Basic JUnit Tests](./ex2)**: Designing a basic class (Calculator) and unit testing its methods.
3. **[Exercise 3: Assertions in JUnit](./ex3)**: Exploring different assertion types such as `assertEquals`, `assertTrue`, `assertFalse`, `assertNull`, and `assertNotNull`.
4. **[Exercise 4: Arrange-Act-Assert (AAA) Pattern and Test Fixtures](./ex4)**: Writing structured tests using `@Before` and `@After` lifecycle methods.
