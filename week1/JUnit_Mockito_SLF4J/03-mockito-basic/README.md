# Booklet 3: Mockito Hands-On Exercises

This booklet covers core mocking concepts using Mockito and JUnit 5:
1. **[Exercise 1: Mocking and Stubbing](./ex1)**: Stubbing external APIs to return predefined values.
2. **[Exercise 2: Verifying Interactions](./ex2)**: Ensuring methods are called with specific parameters.
3. **[Exercise 3: Argument Matching](./ex3)**: Using flexible matchers like `anyString()` to verify behavior.
4. **[Exercise 4: Handling Void Methods](./ex4)**: Working with methods that return nothing.
5. **[Exercise 5: Mocking and Stubbing with Multiple Returns](./ex5)**: Setting up multiple sequential return values.
6. **[Exercise 6: Verifying Interaction Order](./ex6)**: Verifying the sequence of method invocations.
7. **[Exercise 7: Handling Void Methods with Exceptions](./ex7)**: Stubbing void methods to throw exceptions.
