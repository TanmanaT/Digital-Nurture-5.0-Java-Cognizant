# Digital Nurture 5.0 - Java Full Stack Developer Workspace

This repository houses the hands-on training materials, exercises, source code, database scripts, and test suites completed during the training modules.

## Repository Overview

The workspace is organized into weekly directories corresponding to the curriculum schedule:

### Week 1: Core Engineering, PL/SQL and Unit Testing
This folder contains fundamental programming constructs, database operations, and testing frameworks:
* Engineering Concepts: Implementations and tests demonstrating core algorithms, data structures, and classical gang-of-four design patterns (Creational, Structural, and Behavioral patterns).
* PL/SQL Programming: Database programming scripts illustrating stored procedures, control flows, exception handling blocks, custom database functions, explicit cursor processing, auditing triggers, and packaged operations.
* JUnit, Mockito and SL4J: Complete test suites using JUnit 4/5, Mockito for stubbing and verifying external dependencies, Spring Boot testing hooks (such as WebMvcTest, SpringBootTest, DataJpaTest, and MockMvc), and SL4J logging capabilities using standard console and file-based appenders.

### Week 6: Frontend Development
This folder contains the frontend web development modules:
* Angular: Complete frontend applications and components built using the Angular framework.
